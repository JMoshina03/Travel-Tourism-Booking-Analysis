# Travel and Tourism Dashboard Analysis

Internship Project | Excel | SQL | Power BI 

## Project Overview
As part of my internship, I analyzed travel and tourism booking data to understand revenue trends, customer behavior, and booking performance. The dataset was cleaned and structured using SQL, and an interactive dashboard was built in Power BI to present key insights.

## Work Performed
- Cleaned and transformed booking data using MySQL
- Created calculated measures and KPIs in Power BI
- Designed an interactive dashboard with filters and slicers
- Analyzed revenue by destination, season, and travel type
- Evaluated booking channel contribution

## Key Metrics
- Total Bookings: 1000
- Total Travellers: 2217
- Total Revenue: 77.06M
- Average Revenue per Traveler (ARPT): 45.17K

## Insights Observed
- Japan and India contributed the highest share of revenue.
- Family and business travel segments showed strong revenue contribution.
- Agent and mobile app channels played a major role in bookings.
- Seasonal variation influenced overall revenue performance.

## Tools Used
 Microsoft Excel | SQL | Power BI 

## Learning Outcome
This project strengthened my skills in data cleaning, KPI development, dashboard design, and business insight interpretation, preparing me for Business Analyst and Data Analyst roles.
